# Spring Boot und AngularJS

This is sample Spring Boot Single Page Application using Angular. The main features of this proect are [Asset Pipeline] and [Client Dependencies Management using Gradle] integration.

### Prerequisites
- [JDK 1.8]
- [Eclipse Neon]

### How to run 

 - Download/clone.
 - Im Eclipse: Import as Gradle Project
 - DB Setup -> siehe unten
 - gradle clientInstall -> installiert Client Abhängigkeiten unter src/assets/vendor
  - **App.java** as **Java Application** starten. Das startet die Anwendung unter http://localhost:8080
